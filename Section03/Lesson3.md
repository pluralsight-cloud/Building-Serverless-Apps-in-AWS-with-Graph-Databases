## Section 3 Lesson 3

Node.js GitHub resource:
```
https://github.com/nodesource/distributions
```

Install Node.js 19.x:
```
sudo curl -fsSL https://rpm.nodesource.com/setup_19.x | bash -
```

Instal Serverless:
```
sudo npm install -g serverless
```

Deploy Your Service:
```
sls deploy
```

Remove Your Service:
```
sls remove
```
