React Serverless Boilderplate:
```
https://github.com/arabold/serverless-react-boilerplate
```

Add Serverless Webpack:
```
sls plugin install -n serverless-webpack
```

Install Node packages:
```
npm install
```

Update the config file w/ API URL:
```
vi config.js
```

Deploy the UI:
```
sls deploy
```
